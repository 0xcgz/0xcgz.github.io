DarkDork — usage
modules
[ G ] dorking
Set a target domain. Pick a preset. Hit search or copy. Filter by category with the tabs, or build a custom query with the operator buttons.
[ B ] bug bounty
Same flow — but queries hit Wayback, crt.sh, GitHub, and other recon sources instead of Google.
[ O ] osint map
Radial map, 200+ tools across 12 categories. Scroll to zoom, drag to pan. Click any node to open it or expand its branch in the sidebar.
[ + ] custom builder
Open in any dork view. Chain operators token by token — intitle:, inurl:, filetype: etc. Search or copy the full assembled query.
